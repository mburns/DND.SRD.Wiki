# Idle Adventurer

An Idle RPG game for the down-time between dungeon adventures.

## Directories

* `src/` - Godot game source
* `assets/` - multimedia assets (images, fonts, etc)
* `wiki/` - markdown documents of the D&D SRD, adapted from [D&D 5E SRD REmastered]()
